A project to practice Algorithm design and TDD in Java   

## Getting started

### To set up locally

- Clone the repository:
`git clone git@github.com:AJ8GH/java-algorithms.git`

- Build using Maven:
`mvn clean install`

### To run tests

- Use Maven:
`mvn test`

- Or run using TestNG in the IDE 

## Algorithms implemented:
- Find duplicates
- Bubble sort
- Binary search
- Reverse list
- Fibonacci
- Fibonacci recursive
